What is Hive?

Submitted by Mike Combs
on Tue, 11/25/2014 - 18:07

Hive is an open source volunteer project under the Apache Software Foundation. Previously a subproject of Apache Hadoop, it has now graduated to become a top-level project of its own. Hive provides a mechanism to project structure onto this data, and query it using a Structured Query Language (SQL)-like syntax called HiveQL (Hive Query Language). This language also lets traditional map/reduce programmers plug in their custom mappers and reducers when it's inconvenient or inefficient to express this logic in HiveQL.
