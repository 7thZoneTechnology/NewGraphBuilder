SQLstream Leads the Way in Real-time Machine Data Analytics

SQLstream, provider of the real-time data hub for enterprise grade stream processing, today announced the availability of StreamLab 1.1, the next release of the world’s only intelligent guided discovery, streaming analytics and real-time visualization platform. Enterprises using StreamLab are building and deploying massively scalable real-time dashboards from their log and sensor machine data in minutes. The latest release brings faster UI performance, an improved user experience and enhanced visualization capabilities for streaming log and sensor data analytics.

Play demo video: From GPS log data to live vehicle tracking in minutes.

StreamLab is a graphical platform for the intelligent guided discovery, analysis and visualization of log and sensor machine data streams. Users connect to their machine data sources using wizards, and then structure and analyze the streaming data in real-time using interactive graphical tools that offer intelligent suggestions with automated rules. Streaming analytics are connected to real-time dashboards by a simple click. Operational intelligence applications can be built in minutes by analyzing the log data at source, and without having to write SQL or Java analytics, build integration adapters, or even code custom dashboards. StreamLab does it all automatically. StreamLab is built on the award-winning distributed SQL stream processing engine, s-Server, ensuring Big Data scalability, distributed systems operation, platform stability and system reliability.

“StreamLab is a game-changer in the world of machine data analytics and stream processing,” said Edan Kabatchnik, SQLstream’s SVP Products & Engineering. “StreamLab enables enterprises to reduce the cost and complexity of real-time Big Data applications, and deliver operational intelligence applications in minutes.”

New StreamLab features in this release:

  • Instantaneous UI responsiveness through optimization of the auto-generated SQL query
    handling and execution.
  • New log file connection wizard with data visualization for verification of successful connections.
  • Support for additional machine data sources such as web feeds.
  • New dashboard widgets and pan/zoom map displays.
  • New and improved rules for automatic identification of log data attributes such as timestamps.
