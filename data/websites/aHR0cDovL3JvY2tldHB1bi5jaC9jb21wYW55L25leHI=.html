문의하기
회사소개
클라우드 컴퓨팅은 2010년 가트너가 선정한 10대 전략 기술 중 첫 번째로 꼽히는 기술로, 향후 IT 패러다임을 뒤바꿀 것으로 예상하고 있습니다. 또한 Hadoop은 대용량 데이터 저장 및 처리를 위한 오픈소스 플랫폼으로 Yahoo, Facebook, NHN 등에서 활발히 사용되고 있습니다. 넥스알은 이 두 분야의 핵심 플랫폼 S/W 및 응용 서비스를 개발하고 있는 벤쳐기업으로, 국내에서 최고의 기술을 가지고 있다고 자부합니다. 이미 미국, 일본, 중국 등의 관련 커퍼런스에서 회사소개 및 제품 발표를 하여 많은 해외 기업들로부터 인정받았으며 이를 바탕으로 해외 진출을 준비 중에 있습니다.

자유롭고 수평적인 기업문화, 다양한 신기술에 대한 경험을 쌓을 수 있는 점 등은 큰 회사에서는 제공할 수 없는 넥스알의 장점입니다. 이와 같은 장점들 때문에 국내의 대기업에서 이적해 온 분들도 많습니다.

NDAP

NexR Data Analytics Platform (NDAP) is a unified solution that, through combining various Platforms, enables Big Data Analytics in order to solve problems suffered by Enterprises unable to overcome Data Explosions.

In 2011, NDAP has been selected as KT’s BI/DW department’s next-gen analytics system, and is under progressive expansion. In the coming future, it will process unified analysis of all of KT’s data
RHive

RHive is an R package that integrates R environment with Hive, the open source implementation of data warehouse system for Hadoop that facilitates data summarization, ad-hoc queries, and the analysis of big datasets stored in Hadoop compatible file systems.

Using RHive, it is possible to write HQL(Hive Query) in R, launch this query from R, and interact with Hive. Also, R functions and R objects are exported to Hive and launched in Hive via RHive.

In Rhive, the small data is executed in R and the large data is executed in Hive.
