The only Hadoop RDBMS, Splice Machine’s database is designed to scale real-time applications using commodity hardware without application rewrites. The Splice Machine database is a modern, scale-out alternative to traditional RDBMSs, such as Oracle®, MySQL™, IBM DB2® and Microsoft SQL Server®, that can deliver over a 10x improvement in price/performance. As a full-featured RDBMS with ACID transactions and ANSI SQL support, the Splice Machine database helps customers power real-time applications and operational analytics, especially as they approach Big Data scale.

Specialties

Big Data, Hadoop, HBase, SQL, Database, Operational Applications
