DeepDB Scales MySQL for Big Data
Über mich
DeepDB scales MySQL for Big Data. DeepDB delivers accelerated performance and linear scalability for simultaneous data loading, transaction processing and complex queries (all 3 application pillars). DeepDB is a simple and seamless plug-in replacement for InnoDB that preserves application compliance and data integrity while enabling real-time analytics with high-volume data ingestion.
