Why R users should try GraphLab Create

Posted by Chris Dubois on Oct 23, 2014

Devoted R users are getting really excited about GraphLab Create’s scalable data frame, the SFrame. GraphLab Create's Python interface is a small step from the operations that R users are used to - and very little Python knowledge is required. In this blog, I discuss a few areas where GraphLab Create goes beyond the limits of R, like memory and parallelization. I also show a few quick examples of how to use GraphLab Create SFrame commands to illustrate the similarities between pandas and dplyr.
